# 版面模擬（Mockups）

儀表板的 HTML 版面模擬，供 Power BI 實作前討論用（非正式產品）。

## 物料物價風險戰情室_模擬.html

公共工程材料的「物價監看 × 工程成本重估」儀表板模擬，三分頁：

1. **物價戰情** — 風險橫幅、6 品項壓力卡、漲跌幅比較（折線）、誰在漲誰在跌（垂直分歧長條）
2. **工程重估** — 選過去工程 → 用今日物價重算材料成本；含差額、漲價貢獻瀑布、成本走勢、建議編列（範例為鐵路橋梁工程）
3. **品項深掘** — 單品項實際單價走勢 + 各區最新價（垂直長條）

- 圖表數字取自 `data/history/material_price_history.csv` 的真實 12 年資料（民國103-01～115-06），單檔內嵌、可離線開啟。
- 深/淺色主題自適應（右上角切換）。
- 圖表配色經 dataviz 色盲安全驗證；「漲/跌」用語意紅綠、物料識別用類別色。

> ⚠️ 「工程重估」的範例工程用料為示意；正式使用需接入實際工程用料清單（BOQ）或橋梁類型典型配比。
